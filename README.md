# Automated Testing

### Requirements
* NodeJs (vers. 18.3.0 or higher)
* Google Chrome
* Microsoft Edge (Optional)

---

### How to install
1. Clone the repository
2. Navigate to TASK_2 directory
3. Run npm install 

---

### How to run
* npm test (runs on default browser chrome)
* npm test -- --chrome
* npm test -- --edge
